让hexo-blog更像一个blog

hexo自带的blog样本可能不太满足我们对一个blog的追求，所以我们可以对站点配置和主题配置文件进行相应修改

我们先给hexo-选择一个主题

可以从https://github.com/hexojs/hexo/wiki/Themes 选择一个喜欢的主题，到对应github复制地址，git clone到hexo文件夹

git clone git@github.com:litten/hexo-theme-yilia.git themes/yilia

配置文件位置

以上站点配置文件和主题配置文件位置

配置文件配置

  • 主题配置文件

    这里主要改了主题风格和侧边栏显示位置、菜单改成了中文

为菜单加点料

  • hexo文件夹中右击git bash here

  • 依次键入hexo new page tags、hexo new page categories、hexo new page about

  • 以categories为例，修改index.md，type一定要有

  • 这是about的index.md
